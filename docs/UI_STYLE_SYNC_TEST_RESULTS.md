## 測試工具

### 代碼檢查命令

```bash
# 搜索硬編碼的間距
grep -rn "SizedBox(height: [0-9]\|SizedBox(width: [0-9]\|EdgeInsets.all([0-9]" lib/ui/features

# 搜索硬編碼的圓角
grep -rn "BorderRadius.circular([0-9]" lib/ui/features

# 搜索硬編碼的顏色
grep -rn "Color(0x[0-9A-Fa-f]\{6\})" lib/ui/features

# 搜索 Card 的硬編碼樣式
grep -rn "Card(" lib/ui/features | grep -E "color:|shape:|elevation:"

# 搜索 TextField 的硬編碼樣式
grep -rn "TextField(" lib/ui/features -A 10 | grep -E "decoration:.*InputDecoration"
```
