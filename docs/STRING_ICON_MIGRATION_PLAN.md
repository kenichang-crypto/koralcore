# 字符串和圖標遷移計劃

## 目標
1. 一次性將 reef-b-app 中所有未轉換到 koralcore 的字串和多語系轉換過來
2. 一次性將 reef-b-app 的圖標轉換到 koralcore，替換自創的圖標
3. 全部掃描，不遺漏

## 步驟

### 階段 1: 字符串對照和遷移

#### 1.1 提取 reef-b-app 所有字符串
- [ ] 讀取所有語言的 strings.xml
- [ ] 提取所有字符串 key 和值
- [ ] 建立字符串映射表

#### 1.2 對照 koralcore ARB 文件
- [ ] 讀取所有語言的 ARB 文件
- [ ] 找出缺失的字符串
- [ ] 找出不一致的字符串

#### 1.3 添加缺失的字符串
- [ ] 將缺失的字符串添加到所有語言的 ARB 文件
- [ ] 確保使用 reef-b-app 的原始翻譯

### 階段 2: 圖標對照和遷移

#### 2.1 提取 reef-b-app 所有圖標
- [ ] 列出所有 drawable XML 文件
- [ ] 分類圖標（場景、動作、設備等）

#### 2.2 對照 koralcore 使用的圖標
- [ ] 掃描代碼中使用的圖標
- [ ] 找出自創的圖標（Material Icons 等）
- [ ] 找出缺失的圖標

#### 2.3 轉換和替換圖標
- [ ] 將 XML drawable 轉換為 SVG
- [ ] 添加到 assets 目錄
- [ ] 更新代碼使用新的圖標
